The getCldOgImageUrl helper provides a wide range of options for being able to easily generate social cards for Open Graph images.

Configuration for getCldOgImageUrl is the same as [getCldImageUrl](/getcldimageurl/configuration). The only difference is getCldOgImageUrl
provides the following default settings to make generating an OG image URL simpler.

:::tip
To see all available options, visit [getCldImageUrl](/getcldimageurl/configuration).
:::
